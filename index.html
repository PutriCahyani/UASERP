RumahCoding.

Rumah coding menerapkan pembelajaran coding berbasis proyek / project base learning. Pada kursus coding reguler, peserta bisa memilih ingin belajar secara offline atau online. Training akan dipandu oleh trainer secara langsung melalui tatap muka dikelas bagi yang offline, dan untuk yang online peserta akan dipandu oleh trainer melalui conference menggunakan aplikasi zoom atau google meet.

About

Rumah Coding telah banyak memberikan training IT terutama materi coding untuk berbagai macam lembaga baik lembaga pemerintahan maupun lembaga swasta. Kami memberikan jasa corporate training tidak terbatas untuk lembaga di Jakarta saja, tapi mencakup beberapa kota di Indonesia. Materi training yang kami tawarkan sangat beragam mulai dari training android, ios, web design, PHP Code Igniter, PHP Laravel, Java dan lain sebagainya.
